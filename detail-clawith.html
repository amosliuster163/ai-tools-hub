Clawith

桌面AI聊天客户端，支持15+平台、本地知识库RAG、联网搜索、Agent模式，跨平台支持macOS/Windows/Linux。

核心定位

"一个桌面客户端，连接多个AI平台 + 本地知识库"
类似ChatBox/NextChat，但深度集成OpenClaw生态，支持向量数据库RAG和联网搜索增强。

技术架构

Tauri (Rust后端) + React前端，轻量高效，跨平台支持。

Tauri Rust React 向量数据库 MCP协议

支持的AI平台

国内平台 海外平台

主要功能

🌐
多AI平台支持
15+国内外AI平台，统一入口管理
📚
本地知识库(RAG)
集成向量数据库(Milvus/Qdrant/Chroma)，支持私有知识检索
🔍
联网搜索
集成Web搜索增强回答准确性
🤖
Agent模式
支持Agent工具调用，自动化任务执行
🧠
深度思考
支持推理/深度思考模式，复杂问题分析
🔧
MCP工具
支持MCP协议工具集成，扩展能力
📝
每日摘要
定时生成每日摘要，回顾学习内容
💾
聊天历史备份
支持备份和恢复聊天记录

✅ 想要统一桌面客户端管理多个AI账号
✅ 需要本地知识库RAG功能
✅ 想要联网搜索增强
