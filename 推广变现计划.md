### 国际平台申请指南（Impact Radius等）

#### 1. 申请前准备

| 准备项 | 要求 | 建议 |
|--------|------|------|
| **邮箱** | 支持国内邮箱（QQ/163等） | 推荐使用常用邮箱 |
| **网站** | 需有真实流量网站 | 你的AI工具集网站已满足 |
| **收款账户** | PayPal或国际银行账户 | 建议注册PayPal |
| **网络环境** | 可能需要国际网络 | 使用代理/VPN |

#### 2. Impact Radius申请步骤

```
步骤1: 注册账号
├─ 访问 impact.com
├─ 点击 "For Media Partners"（媒体合作伙伴）
├─ 选择 "Sign Up" 或 "Get Started"
├─ 填写邮箱（国内邮箱可用）
├─ 设置密码

步骤2: 填写媒体资料
├─ 网站URL: https://amosliuster163.github.io/ai-tools-hub/
├─ 网站类型: Technology / Software
├─ 流量说明: 写"公众号推广导流，月访问量XXX"
├─ 推广方式: Content Marketing, Blog Reviews

步骤3: 等待审核
├─ 通常1-3个工作日
├─ 可能需要补充流量证明
├─ 公众号粉丝数可作为证明

步骤4: 申请具体商户
├─ 搜索 Grammarly
├─ 申请推广计划
├─ 获取推广链接
```
