🔧 Claude Code API Proxy

Anthropic API → OpenAI兼容格式转换器，让Claude Code通过OpenAI SDK调用Claude模型

开发工具 API代理 TypeScript 开源

📌 项目简介

Claude Code Proxy是一个API代理服务，将Anthropic Claude API转换为OpenAI兼容格式。这使得：

  • 使用Claude Code CLI工具的开发者可以无缝切换模型
  • 现有OpenAI SDK代码可以直接调用Claude模型
  • 多模型应用可以统一API接口

🎯 核心功能

  • API转换 - Anthropic格式 → OpenAI格式
  • 流式响应 - 支持SSE流式输出
  • 模型映射 - claude-3-5-sonnet → gpt-4等
  • 轻量部署 - TypeScript实现，可本地运行

🛠️ 技术栈

📘 TypeScript
🟢 Node.js
⚡ Express
🔑 Anthropic SDK

🚀 快速开始

# 克隆项目
git clone https://github.com/instructkr/claude-code.git
cd claude-code

# 安装依赖
npm install

# 配置环境变量
export ANTHROPIC_API_KEY=your_key_here

# 启动代理服务
npm run start

# 代理地址
http://localhost:8080/v1

💡 使用场景

场景 说明
Claude Code CLI 设置OPENAI_BASE_URL指向代理地址
现有OpenAI代码 无需修改代码，直接调用Claude
多模型切换 同一接口支持不同后端模型
