📖 项目简介

Claude Code API Proxy 是一个将 Anthropic Claude API 转换为 OpenAI 兼容格式的代理服务器。 它允许你使用 OpenAI SDK 或任何兼容 OpenAI API 的工具来调用 Claude 模型， 无需修改现有代码即可享受 Claude 的强大能力。

💡 核心功能

  • API格式转换：将 Anthropic API 请求/响应转换为 OpenAI 格式
  • Claude Code支持：完整支持 Claude Code 工具调用功能
  • 多端点代理：支持 Chat、Messages 等多个 API 端点
  • TypeScript实现：类型安全、易于扩展

🚀 快速开始

# 克隆仓库
git clone https://github.com/instructkr/claude-code.git

# 安装依赖
npm install

# 配置 Anthropic API Key
export ANTHROPIC_API_KEY=your_api_key

# 启动代理服务器
npm run start

# 现在可以使用 OpenAI SDK 调用 Claude！
