## 🦐 OpenClaw生态工具

### ClawdHome - Mac多实例管理控制台

**定位**: 在单台Mac上安全运行多个独立OpenClaw实例

**核心功能**:
- 多实例隔离（每个有自己的账号、数据、权限）
- 克隆Shrimp实验新配置
- Gateway生命周期管理
- 图形界面管理

**适合**: 想在一台Mac跑多个独立Agent的用户

**链接**: https://github.com/ThinkInAIXYZ/clawdhome

---

### Clawith - 桌面AI聊天客户端

**定位**: 一个客户端连接多个AI平台 + 本地知识库

**核心功能**:
- 15+ AI平台（通义、智谱、DeepSeek、Claude、GPT等）
- 本地知识库RAG（向量数据库）
- 联网搜索增强
- Agent模式 + MCP工具

**适合**: 统一管理多AI账号、需要知识库RAG的用户

**链接**:
- 官网下载: https://clawith.clawd.ai
- GitHub: https://github.com/dataelement/Clawith
