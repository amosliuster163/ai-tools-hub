📊 OpenClaw Web Dashboard

OpenClaw Gateway的可视化管理界面，实时监控智能体状态

开发工具 React TypeScript Tailwind CSS 开源

📌 项目简介

OpenClaw Web Dashboard是为OpenClaw Gateway开发的Web可视化界面，旨在降低CLI门槛，让普通用户也能轻松管理和监控OpenClaw智能体。

项目采用现代前端技术栈：React 18 + TypeScript + Vite + Tailwind CSS，提供响应式设计和实时状态更新。

🎯 核心功能

  • 首页仪表盘 - 显示Gateway健康状态、运行时间、模型信息
  • Skills管理 - 查看已安装Skills列表，启用/禁用控制
  • Sessions管理 - 查看活跃会话列表，会话状态监控
  • 配置管理 - 查看和修改Gateway配置

🛠️ 技术栈

⚛️ React 18
📘 TypeScript
⚡ Vite
🎨 Tailwind CSS
🧭 React Router
💎 Lucide Icons

🚀 快速开始

# 克隆项目
git clone https://github.com/amosliuster163/openclaw-dashboard.git
cd openclaw-dashboard

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

💡 用户痛点

OpenClaw是一个强大的AI智能体框架，但存在以下使用门槛：

  • CLI命令行操作对普通用户不够友好
  • 无法直观查看智能体运行状态
  • 50+ GitHub Issues需求可视化界面
  • 难以监控多会话并行执行情况

Web Dashboard正是为解决这些问题而生！

🔗 Gateway API

Dashboard通过以下API与OpenClaw Gateway通信：

端点 功能
/health 健康状态检查
gateway config.get 获取配置信息
sessions list 会话列表查询
openclaw status 系统状态查看
