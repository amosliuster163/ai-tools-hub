OpenClaw一键备份

专为OpenClaw用户设计的自动化备份脚本，一键备份配置、工作区、向量记忆等核心数据，支持定时自动备份和一键恢复。

核心定位

"5秒完成OpenClaw完整备份"
OpenClaw一键备份脚本是一个Shell脚本工具，自动备份OpenClaw的所有核心数据，包括配置文件、工作区记忆、向量索引、认证信息等。支持手动执行和定时自动备份，备份文件可上传至GitHub私有仓库实现云端备份。

备份内容

数据类型 路径 说明
配置文件 openclaw.json OpenClaw主配置
工作区 workspace/ MEMORY.md、记忆文件等
向量记忆 memory/*.sqlite 向量检索索引数据库
认证信息 auth-profiles.json 各Agent认证配置
插件配置 plugins/ 已安装插件目录

快速使用

# 下载脚本
curl -fsSL https://raw.githubusercontent.com/amosliuster163/openclaw-backup/main/openclaw-backup.sh -o ~/openclaw-backup.sh
chmod +x ~/openclaw-backup.sh

# 执行备份
~/openclaw-backup.sh

# 查看备份
ls -lh ~/OpenClaw-Backups/

# 设置每日自动备份（凌晨2点）
(crontab -l 2>/dev/null; echo "0 2 * * * ~/openclaw-backup.sh") | crontab -

# 恢复备份
tar -xzf ~/OpenClaw-Backups/openclaw-backup-YYYYMMDD_HHMMSS.tar.gz
cp -r openclaw-backup-*/* ~/.openclaw/
launchctl start com.openclaw.gateway  # macOS重启服务
