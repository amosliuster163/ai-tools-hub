Coding Plan额度用完怎么办？意外发现免费推理模型！

你用OpenClaw的Coding Plan套餐，突然收到提示"额度快用完了"。怎么办？

今天记录完整的备用API配置过程，还有一个意外发现：Coding Plan其实也有推理模型，免费使用！

配置备用API

在models.providers里新增aliyun-payg：

"aliyun-payg": {
  "baseUrl": "https://dashscope.aliyuncs.com/compatible-mode/v1",
  "apiKey": "sk-你的API密钥",
  "api": "openai-completions",
  "models": [...]
}
踩坑点：API Key必须带 sk- 前缀！缺少前缀会认证失败。

两个API的区别

对比项Coding Plan按量付费API

意外发现：Coding Plan也有推理模型！

测试"解释WAL协议"，观察API返回的 reasoning_tokens：

模型API来源推理tokens推理能力
GLM-5Coding Plan786带推理
qwen3.6-plus备用API157带推理
Qwen3-MaxCoding Plan无无推理
GLM-5的推理tokens是786，比qwen3.6-plus的157还多！Coding Plan的GLM-5是智谱最新推理模型，免费使用！
